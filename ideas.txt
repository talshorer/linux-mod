ksignal (kthread, debugfs)
	export a debugfs file to spawn kthreads. these will sleep in a loop, waking
	up when they receive a signal. they will log the signal and in case it's
	not SIGTERM go to sleep again.
virtusb kit
	virtudc (usb gadget framework)
		register a virtual usb to which gadget drivers can connect.
	g_hub (usb gadget)
		spawns many virtudc and pipes them to the host computer.
	virthcd (usb host framework)
		register a virtual usb host controller to which gadgets connected to
		virtudc can connect, creating a complete virtual usb system.
